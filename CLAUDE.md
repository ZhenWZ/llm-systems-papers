# Claude Instructions

Follow the repository rules in [AGENTS.md](AGENTS.md).

This repository is a Chinese-first LLM systems paper library. When updating notes, keep professional terms in English and preserve the standard note sections:

- Metadata
- 一句话结论
- Related Works
- 问题与背景
- 方法与系统设计
- 创新点
- 实验与结论
- 局限性与风险
- 对 LLM Systems 的启发
- 复现/阅读建议

When adding papers, update `README.md`, the relevant `categories/*.md` page, and `data/papers.yml`.
